Matplotlib, NumPy et SciPy pour le Calcul Scientifique#

Ce document propose une introduction aux bibliothèques Matplotlib, NumPy, et SciPy, essentielles pour le calcul scientifique et la visualisation en Python. Il se termine par une section sur les bonnes pratiques et une série d’exercices pour consolider les connaissances.

Matplotlib : Création de Visualisations#

Introduction à Matplotlib#

Matplotlib est une bibliothèque de traçage 2D très utilisée pour créer des graphiques en Python. Elle permet de réaliser des visualisations simples ou complexes, adaptées aux besoins de l’analyse scientifique.

# Installation de Matplotlib
# !pip install matplotlib

# Importation standard
import matplotlib.pyplot as plt

Création de Graphiques Simples#

# Création d'un graphique linéaire
x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

plt.plot(x, y, marker='o', linestyle='-', color='b', label="y = x²")
plt.title("Exemple de Graphique")
plt.xlabel("x")
plt.ylabel("y")
plt.legend()
plt.grid(True)
plt.show()

Visualisations Avancées#

# Histogramme
data = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
plt.hist(data, bins=4, color='green', edgecolor='black')
plt.title("Histogramme")
plt.xlabel("Valeurs")
plt.ylabel("Fréquence")
plt.show()

Principales Fonctions de Matplotlib#

Fonctionnalité Description Exemple
Création d'un graphique simple Tracer un graphique de base avec des points ou des lignes. 

import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 35]


plt.plot(x, y, label=”Ligne 1”)
plt.xlabel(“Axe X”)
plt.ylabel(“Axe Y”)
plt.title(“Graphique Simple”)
plt.legend()
plt.show()
Styles de tracés Personnaliser les lignes : couleurs, styles, marqueurs. 

x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 35]


plt.plot(x, y, “r–”, marker=”o”, label=”Ligne rouge pointillée”)
plt.xlabel(“Axe X”)
plt.ylabel(“Axe Y”)
plt.title(“Styles de Tracés”)
plt.legend()
plt.show()
Barres Tracer des diagrammes en barres verticales ou horizontales. 

categories = [“A”, “B”, “C”]
values = [10, 15, 7]


plt.bar(categories, values, color=”blue”)
plt.title(“Diagramme en Barres”)
plt.show()
Histogrammes Représenter la distribution des données avec des histogrammes. 

import numpy as np


data = np.random.randn(1000)
plt.hist(data, bins=20, color=”green”, edgecolor=”black”)
plt.title(“Histogramme”)
plt.xlabel(“Valeurs”)
plt.ylabel(“Fréquence”)
plt.show()
Nuages de points Tracer des données comme points dispersés. 

x = [5, 7, 8, 7, 2, 17, 2, 9, 4, 11]
y = [99, 86, 87, 88, 100, 86, 103, 87, 94, 78]


plt.scatter(x, y, color=”purple”)
plt.title(“Nuage de Points”)
plt.xlabel(“Axe X”)
plt.ylabel(“Axe Y”)
plt.show()
Graphiques multiples Tracer plusieurs graphiques côte à côte ou empilés. 

x = [1, 2, 3, 4, 5]
y1 = [1, 4, 9, 16, 25]
y2 = [25, 16, 9, 4, 1]


plt.subplot(1, 2, 1)
plt.plot(x, y1, “b-“)
plt.title(“Graphique 1”)


plt.subplot(1, 2, 2)
plt.plot(x, y2, “r-“)
plt.title(“Graphique 2”)


plt.tight_layout()
plt.show()
Graphiques en 3D Tracer des graphiques dans un espace tridimensionnel. 

from mpl_toolkits.mplot3d import Axes3D
import numpy as np


fig = plt.figure()
ax = fig.add_subplot(111, projection=”3d”)
x = np.random.rand(50)
y = np.random.rand(50)
z = np.random.rand(50)


ax.scatter(x, y, z, c=”r”, marker=”o”)
ax.set_xlabel(“X”)
ax.set_ylabel(“Y”)
ax.set_zlabel(“Z”)
plt.show()

NumPy : Calcul Numérique Performant#

Introduction à NumPy#

NumPy (Numerical Python) est une bibliothèque dédiée au calcul scientifique. Elle offre :

  • Des tableaux multidimensionnels performants.

  • Des opérations mathématiques vectorisées.

  • Des outils pour l’algèbre linéaire et les statistiques.

# Installation et importation
!pip install numpy
import numpy as np

Requirement already satisfied: numpy in d:\pedagogie\estc\jupyter-book-env\lib\site-packages (2.1.3)

Création et Manipulation de Tableaux#

# Création de tableaux
tableau_1d = np.array([1, 2, 3, 4, 5])
tableau_2d = np.array([[1, 2, 3], [4, 5, 6]])

# Propriétés des tableaux
print(f"Forme : {tableau_2d.shape}, Type : {tableau_2d.dtype}, Dimensions : {tableau_2d.ndim}")

Forme : (2, 3), Type : int64, Dimensions : 2

Opérations Mathématiques#

# Opérations vectorisées
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(a + b)          # Addition
print(a ** 2)         # Puissance
print(np.mean(a))     # Moyenne

[5 7 9]
[1 4 9]
2.0

Algèbre Linéaire#

# Multiplication matricielle et inverse
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
print(np.dot(A, B))   # Produit matriciel
print(np.linalg.inv(A)) # Inverse

[[19 22]
 [43 50]]
[[-2.   1. ]
 [ 1.5 -0.5]]

Principales Fonctions de Numpy#

Fonctionnalité Description Exemple
Création de tableaux Créer des tableaux 1D, 2D ou spéciaux (zeros, ones, identity). 

import numpy as np
arr = np.array([1, 2, 3])
zeros = np.zeros((2, 2))
print(arr)
print(zeros)
Accès aux éléments Accéder aux données via index ou slicing. 

tab = np.array([[1, 2], [3, 4]])
print(tab[0, 1])  # Ligne 0, colonne 1
print(tab[:, 0])  # Première colonne
Opérations mathématiques Opérations élément par élément. 

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(a + b)      # Addition
print(np.sqrt(a)) # Racine carrée
Statistiques Calculs : somme, moyenne, écart-type, etc. 

arr = np.array([1, 2, 3, 4, 5])
print(np.mean(arr))  # Moyenne
print(np.std(arr))   # Écart-type
Algèbre linéaire Inverses, déterminants, etc. 

A = np.array([[1, 2], [3, 4]])
inv_A = np.linalg.inv(A)
print(inv_A)

SciPy : Calculs Avancés#

Introduction à SciPy#

SciPy est une bibliothèque construite sur NumPy, offrant des fonctionnalités supplémentaires pour :

  • L’optimisation.

  • L’intégration.

  • Le traitement de signaux.

# Installation et importation
#!pip install scipy
#from scipy import linalg, optimize

Exemple d’Optimisation#

# Trouver un minimum local
def f(x):
    return x**2 + 4*np.sin(2*x)

resultat = optimize.minimize(f, x0=1)
print(f"Minimum local trouvé : {resultat.x}")

Résolution de Systèmes Linéaires#

# Résolution avec SciPy
A = np.array([[3, 2], [1, 4]])
b = np.array([6, 8])
solution = linalg.solve(A, b)
print(f"Solution : {solution}")

Principales Fonctions de SciPy#

Fonctionnalité Description Exemple
Résolution de systèmes linéaires Résoudre un système d'équations linéaires \(Ax = b\). 

import numpy as np
from scipy.linalg import solve

A = np.array([[3, 2], [1, 4]])
b = np.array([6, 8])


x = solve(A, b)
print(x)
Calcul des valeurs propres Calculer les valeurs propres et les vecteurs propres d’une matrice. 

from scipy.linalg import eig


A = np.array([[5, 4], [2, 1]])
vals, vecs = eig(A)


print(« Valeurs propres: », vals)
print(« Vecteurs propres: », vecs)
Interpolation Interpoler une fonction à partir de points de données. 

from scipy.interpolate import interp1d
import matplotlib.pyplot as plt


x = [0, 1, 2, 3, 4]
y = [0, 2, 4, 6, 8]


f = interp1d(x, y, kind=”linear”)
x_new = np.linspace(0, 4, 50)
y_new = f(x_new)


plt.plot(x, y, “o”, label=”Points”)
plt.plot(x_new, y_new, “-”, label=”Interpolation”)
plt.legend()
plt.show()
Optimisation Trouver le minimum ou maximum d’une fonction. 

from scipy.optimize import minimize


def func(x):
return x**2 + 5 * np.sin(x)


result = minimize(func, x0=2)
print(« Minimum trouvé à: », result.x)
Transformée de Fourier Calculer la transformée de Fourier d’un signal. 

from scipy.fft import fft, fftfreq
import numpy as np


t = np.linspace(0, 1, 500, endpoint=False)
signal = np.sin(2 * np.pi * 50 * t)


fft_result = fft(signal)
frequencies = fftfreq(len(t), d=t[1] - t[0])


print(« Transformée de Fourier: », fft_result)
Traitement d’image (Filtrage Gaussien) Appliquer un filtre gaussien à une image. 

from scipy.ndimage import gaussian_filter
import matplotlib.pyplot as plt


image = np.random.rand(100, 100)
filtered_image = gaussian_filter(image, sigma=2)


plt.subplot(1, 2, 1)
plt.title(« Image originale »)
plt.imshow(image, cmap=”gray”)


plt.subplot(1, 2, 2)
plt.title(« Image filtrée »)
plt.imshow(filtered_image, cmap=”gray”)
plt.show()
Statistiques (Tests t) Effectuer un test t pour comparer deux ensembles de données. 

from scipy.stats import ttest_ind


data1 = [5.3, 7.1, 8.5, 6.7, 5.9]
data2 = [5.8, 7.4, 8.1, 6.6, 6.2]


t_stat, p_value = ttest_ind(data1, data2)
print(« t-statistique: », t_stat)
print(« p-valeur: », p_value)

Bonnes Pratiques#

  • Clarté du Code : Utilisez des noms de variables significatifs.

  • Visualisations Lisibles : Ajoutez des titres, des légendes, et des axes clairs à vos graphiques.

  • Gestion des Erreurs : Vérifiez les dimensions et les types de données avant d’effectuer des opérations.
